# 📊 DISPATCH MANAGEMENT DASHBOARD

## 🧩 1. OVERVIEW SECTION (Top Summary Cards)

These are **real-time KPI cards** displayed at the top.

### 🔹 Metrics Cards

#### 1. Total Dispatched Today

* **Definition:** Number of invoices marked as *dispatched* today
* **Formula:**

  ```
  count(dispatch where date = today)
  ```
* **UI:**

  * Big number (e.g., 45)
  * Small label: “+12% from yesterday”

---

#### 2. Pending Dispatch

* **Definition:** Invoices assigned to dispatch but not yet shipped
* **Includes statuses:**

  * `pending_dispatch`
  * `packing`
* **Formula:**

  ```
  count(invoices where status IN (pending_dispatch, packing))
  ```

---

#### 3. Delivered Orders

* **Definition:** Successfully delivered packages
* **Condition:**

  ```
  status = delivered AND delivery_confirmed = true
  ```

---

#### 4. Failed / Damaged Deliveries

* **Definition:** Orders marked damaged or incomplete
* **Conditions:**

  * `delivery_status = damaged`
  * OR `all_items_received = false`

---

## 🚚 B. Courier Performance

### Type:

* Bar chart or leaderboard table

### Metrics per courier:

* Total deliveries
* Success rate (%)
* Failed deliveries
* Avg delivery time

### Example:

| Courier       | Delivered | Failed | Success % |
| ------------- | --------- | ------ | --------- |
| Fargo Courier | 120       | 5      | 96%       |
| DHL           | 80        | 10     | 88%       |

---

### Formula:

```id="courier_perf"
success_rate = (delivered / total_assigned) * 100
```

---

## ⏱️ C. Average Delivery Time

### Type:

* Line chart or KPI card

### Formula:

```id="avg_time"
delivery_time = delivery_time - dispatch_time
```

### Output:

* Example:

  * “Avg Delivery Time: 6h 25min”

### Breakdown:

* Per courier
* Per region (optional)
